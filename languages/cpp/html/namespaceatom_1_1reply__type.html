◆ flat_response

using atom::reply_type::flat_response = typedef std::pair<std::shared_ptr<const char *>, size_t>

flat response: holds replies for XADD, XDEL, SET, XACK, etc. The shared pointer points to buffer where the data begins, and size_t indicates the number of characters in the data.

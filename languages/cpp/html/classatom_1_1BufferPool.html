Constructor for BufferPool, which holds a maximum of 20 buffers that can be used concurrently for separate read/write operations. This will throw if buffers_requested is greater than 20.

Template Parameters
bufferrefers to the underlying buffer type, i.e. boost::asio::streambuf.
Parameters
buffers_requestedthe number of buffers to initialize BufferPool with
timeouttimeout in milliseconds to wait for a buffer to become available for use

Decrement a reference to a buffer instance. After releasing a buffer, that buffer is not guaranteed to be available for use, and the user must call get_buffer() to get a buffer to work with.

Parameters
